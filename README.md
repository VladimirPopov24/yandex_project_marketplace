# Modex Marketplace

Полноценный маркетплейс на FastAPI с поддержкой ролей покупатель / продавец / администратор.

[Пояснительная записка](ПОЯСНИТЕЛЬНАЯ_ЗАПИСКА.md)

## Стек технологий

| Слой | Технология |
|------|-----------|
| Backend | FastAPI, Uvicorn |
| База данных | SQLite + SQLAlchemy ORM |
| Аутентификация | JWT (python-jose), Werkzeug (хеширование) |
| Шаблоны | Jinja2 |
| Загрузка файлов | python-multipart |

## Возможности

### Покупатель
- Просмотр товаров с фильтрацией (поиск, категория, диапазон цен, сортировка)
- Детальная страница товара с похожими товарами
- Корзина с управлением количеством
- Оформление заказа
- Избранное / вишлист
- Личный кабинет
- Возможность стать продавцом

### Продавец
- Дашборд с собственными товарами
- Добавление / редактирование / удаление товаров
- Загрузка фото товара
- Отслеживание статуса модерации (pending → approved / rejected)

### Администратор
- Статистика по платформе
- Очередь модерации товаров (одобрить / отклонить / удалить)
- Управление пользователями (поиск, блокировка, смена роли)
- Управление категориями

## Структура проекта

```
yandex_project_marketplace/
├── requirements.txt
├── modex.db                    # SQLite база данных
└── app/
    ├── main.py                 # Точка входа, регистрация роутеров, seed-данные
    ├── models.py               # ORM-модели (User, Product, CartItem, Order, …)
    ├── database.py             # Подключение к SQLite
    ├── auth.py                 # JWT, хеширование паролей, текущий пользователь
    ├── routers/
    │   ├── auth.py             # Логин, регистрация, выход
    │   ├── products.py         # Список и детали товаров
    │   ├── cart.py             # Корзина и оформление заказа
    │   ├── favorites.py        # Избранное
    │   ├── seller.py           # Дашборд продавца
    │   └── admin.py            # Панель администратора
    ├── templates/              # Jinja2-шаблоны
    │   ├── seller/
    │   └── admin/
    └── static/
        └── uploads/            # Загружаемые изображения товаров
```

## Установка и запуск

**1. Клонировать репозиторий и перейти в папку**
```bash
git clone <url>
cd yandex_project_marketplace
```

**2. Создать виртуальное окружение и установить зависимости**
```bash
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

**3. Запустить сервер**
```bash
uvicorn app.main:app --reload
```

Приложение будет доступно по адресу **http://localhost:8000**

> При первом запуске база данных создаётся автоматически и заполняется тестовыми данными (40+ товаров, 14 категорий).

## Основные маршруты

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/` | Главная страница |
| GET/POST | `/auth/login` | Вход |
| GET/POST | `/auth/register` | Регистрация |
| GET | `/products` | Каталог товаров |
| GET | `/products/{id}` | Страница товара |
| GET | `/cart` | Корзина |
| GET | `/cart/checkout` | Оформление заказа |
| GET | `/favorites` | Избранное |
| GET | `/seller/dashboard` | Дашборд продавца |
| GET | `/admin/dashboard` | Панель администратора |
| GET | `/docs` | Авто-документация API (Swagger) |

## Модели данных

- **User** — покупатель / продавец / admin, блокировка
- **Product** — товар с ценой, категорией, изображением и статусом модерации
- **Category** — категория товаров
- **CartItem** — элемент корзины с количеством
- **Favorite** — избранный товар пользователя
- **Order** — заказ с итоговой суммой и статусом
